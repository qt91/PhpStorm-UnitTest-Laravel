# Laravel setup run unit test in PHPStorm
User can use phpstorm for faster run any unit test
## How to run docker
```
   $ make start
```
## Access to docker phpfpm
```
   $ make docker_php
```
## Access to site
```
http://localhost:8000/
```

## Project note
* Use PHP Standard Recommendation PSR-2
* Quality Tool is PHP Mess Detector (phpmd v-2.6)
